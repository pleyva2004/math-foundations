# Tour: CS Undergrad

> "I've had calc 1 and linear algebra. I want to read ML papers."

Skip the absolute basics. Hits the calc → probability → info-theory pipeline that every modern ML paper assumes.

## The path

1. [`12-derivatives`](../concepts/12-derivatives/README.md), [`13-multivariate-calculus`](../concepts/13-multivariate-calculus/README.md), [`14-gradient-jacobian`](../concepts/14-gradient-jacobian/README.md) — refresh.
2. [`15-integration`](../concepts/15-integration/README.md) — the Riemann integral; you'll need it for expectations.
3. [`19-probability-measures`](../concepts/19-probability-measures/README.md), [`20-conditional-probability`](../concepts/20-conditional-probability/README.md), [`21-independence`](../concepts/21-independence/README.md).
4. [`22-random-variables`](../concepts/22-random-variables/README.md), [`23-distributions`](../concepts/23-distributions/README.md), [`24-pdf`](../concepts/24-pdf/README.md).
5. [`25-expectation`](../concepts/25-expectation/README.md), [`26-variance-covariance`](../concepts/26-variance-covariance/README.md).
6. [`28-change-of-variables-probability`](../concepts/28-change-of-variables-probability/README.md) — change-of-variables formula. Essential for flow-based models.
7. [`36-shannon-entropy`](../concepts/36-shannon-entropy/README.md), [`37-cross-entropy`](../concepts/37-cross-entropy/README.md), [`38-kl-divergence`](../concepts/38-kl-divergence/README.md) — the loss-function trinity.

You can now read most NeurIPS / ICML papers. To go deeper into diffusion / flow / RL: [`tours/math-grad.md`](math-grad.md).
