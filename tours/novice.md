# Tour: Novice

> "I don't know what a function is."

Linear walk from absolute zero to the first applied probability concepts. Allow ~10 hours total; each concept ~15-20 minutes.

## The path

1. [`00-sets-and-functions`](../concepts/00-sets-and-functions.md) — what a set is, what a function is, why both matter.
2. [`01-logic-and-proof`](../concepts/01-logic-and-proof.md) — propositions, truth, the four proof strategies.
3. [`02-counting`](../concepts/02-counting.md) — permutations, combinations, the pigeonhole principle.
4. [`03-relations-and-orderings`](../concepts/03-relations-and-orderings.md) — equivalence, partial order.

(Pause and rest. The next chunk is harder.)

5. [`04-groups-rings-fields`](../concepts/04-groups-rings-fields.md) — what an "algebraic structure" is.
6. [`05-vector-spaces`](../concepts/05-vector-spaces.md) — the universal object of linear algebra.
7. [`06-linear-maps`](../concepts/06-linear-maps.md) — matrices as encoded linear maps.
8. [`09-norms-and-metrics`](../concepts/09-norms-and-metrics.md) — distance, length, the building blocks of analysis.

(You now know enough to read any technical-paper introduction.)

9. [`17-sample-spaces`](../concepts/17-sample-spaces.md) — the starting point of probability.
10. [`19-probability-measures`](../concepts/19-probability-measures.md) — what a "probability" actually is.
11. [`22-random-variables`](../concepts/22-random-variables.md) — putting numbers on random outcomes.
12. [`25-expectation`](../concepts/25-expectation.md) — the average.

At this point, branch into [`tours/cs-undergrad.md`](cs-undergrad.md) (which assumes everything above).
