# KL Divergence

**Level:** `advanced`
**Concept ID:** `38-kl-divergence`
**Prerequisites:** [37-cross-entropy](../37-cross-entropy/README.md)

## Plain-English intro
D_KL(p || q) = E_p[log p - log q]; the relative entropy / information gain in moving from q to p. Non-symmetric, non-metric.
