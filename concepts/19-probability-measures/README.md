## Plain-English Exposition

A **probability measure** is the mathematical object that turns the informal phrase "the probability of event $A$" into a precise number. Given a sample space $\Omega$ (the set of all possible outcomes) and a $\sigma$-algebra $\mathcal{F}$ of measurable events (node 18), a probability measure $P$ is a *rule* that assigns each event $A \in \mathcal{F}$ a number $P(A) \in [0,1]$ in a way that is internally consistent: the empty event has probability zero, the whole sample space has probability one, and disjoint events combine by addition. These constraints are encoded in the **Kolmogorov axioms**, which since 1933 have served as the bedrock of modern probability theory. Every probabilistic claim — whether about a coin flip, a Markov chain, or the output distribution of a language model — is, when made rigorous, a statement about some probability measure on some measurable space.

## Formal Definitions

A **probability space** is a triple $(\Omega, \mathcal{F}, P)$ where:

- $\Omega$ is a non-empty set, the **sample space**;
- $\mathcal{F} \subseteq 2^{\Omega}$ is a $\sigma$-algebra on $\Omega$;
- $P : \mathcal{F} \to [0,1]$ is a **probability measure**, i.e. a set function satisfying the three Kolmogorov axioms:

**(K1) Non-negativity.** $P(A) \geq 0$ for every $A \in \mathcal{F}$.

**(K2) Normalisation.** $P(\Omega) = 1$.

**(K3) Countable additivity ($\sigma$-additivity).** For every countable collection $\{A_n\}_{n \geq 1} \subset \mathcal{F}$ of pairwise disjoint events,
$$P\!\left(\bigcup_{n=1}^{\infty} A_n\right) = \sum_{n=1}^{\infty} P(A_n).$$

From these three axioms one derives every standard rule of probability, including $P(\emptyset) = 0$, finite additivity, monotonicity ($A \subseteq B \Rightarrow P(A) \leq P(B)$), the complement rule $P(A^c) = 1 - P(A)$, and the bounds $0 \leq P(A) \leq 1$.

## Core Theorems

**Theorem (Continuity from below).** If $A_1 \subseteq A_2 \subseteq \cdots$ is an increasing sequence in $\mathcal{F}$ and $A = \bigcup_n A_n$, then $P(A_n) \uparrow P(A)$.

*Sketch.* Set $B_1 = A_1$, $B_n = A_n \setminus A_{n-1}$ for $n \geq 2$. The $B_n$ are pairwise disjoint with $\bigcup B_n = A$ and $A_n = \bigsqcup_{k\leq n} B_k$. By (K3), $P(A) = \sum_{n=1}^{\infty} P(B_n) = \lim_n \sum_{k=1}^n P(B_k) = \lim_n P(A_n)$.

**Theorem (Inclusion–exclusion, two-set form).** For any $A, B \in \mathcal{F}$,
$$P(A \cup B) = P(A) + P(B) - P(A \cap B).$$
A full proof is in `lesson.tex`. The general $n$-set form follows by induction.

**Theorem (Boole's inequality / countable subadditivity).** For any countable collection $\{A_n\} \subseteq \mathcal{F}$,
$$P\!\left(\bigcup_n A_n\right) \leq \sum_n P(A_n).$$

*Sketch.* Disjointify: let $B_1 = A_1$ and $B_n = A_n \setminus \bigcup_{k<n} A_k$. Then $B_n \subseteq A_n$ so $P(B_n) \leq P(A_n)$, and the $B_n$ partition $\bigcup A_n$, so $P(\bigcup A_n) = \sum P(B_n) \leq \sum P(A_n)$. Boole is the workhorse of union-bound arguments throughout learning theory.

## Worked Examples

**Example 1: Fair die.** Let $\Omega = \{1, 2, 3, 4, 5, 6\}$ and $\mathcal{F} = 2^{\Omega}$. Define $P(A) = |A|/6$. Axioms (K1)–(K3) are immediate, and $P(\{1,3,5\}) = 1/2$ recovers "probability of an odd roll is one half." More generally, any weight vector $(w_1, \dots, w_6)$ with $w_i \geq 0$ and $\sum w_i = 1$ defines a probability measure via $P(A) = \sum_{i \in A} w_i$ — this is the family `code.py` implements.

**Example 2: Uniform measure on $[0,1]$.** Take $\Omega = [0,1]$, $\mathcal{F}$ the Borel $\sigma$-algebra, and $P = \lambda$ the restriction of Lebesgue measure. Then $P([0, 1/2]) = 1/2$ and $P(\{x\}) = 0$ for every singleton — the first example where uncountably many outcomes each get zero probability yet their union has probability one. This is the canonical illustration of why $\sigma$-additivity, not merely finite additivity, is the right axiom.

## Why This Matters for ML and Papers

Every "probability" you read in a research paper is, when unrolled, a probability measure on some measurable space. The training distribution is a measure on token sequences; the policy is a conditional measure on actions; concentration inequalities like Hoeffding and Bernstein are statements about how much mass a measure can place far from its mean. In **GRPO** (Akgül et al. 2026, §3.1), advantage normalisation treats reward correctness as a Bernoulli measure on $\{0,1\}$: $P(\text{correct}) = \mu$, $P(\text{incorrect}) = 1-\mu$, and the centred advantage $r - \mu$ has expectation zero precisely because $P$ is normalised (K2). Without (K1)–(K3) the variance estimate that stabilises the policy gradient would be undefined.

## Try It Yourself — Exercises

1. **(Complement and monotonicity.)** Prove from the axioms that $P(A^c) = 1 - P(A)$ and that $A \subseteq B$ implies $P(A) \leq P(B)$. *Hint:* write $\Omega = A \sqcup A^c$ and $B = A \sqcup (B \setminus A)$, then apply (K2) and (K3).
2. **(Continuity from above.)** Show that if $A_1 \supseteq A_2 \supseteq \cdots$ and $A = \bigcap_n A_n$, then $P(A_n) \downarrow P(A)$. *Hint:* apply continuity from below to the complements $A_n^c$ and use the complement rule.
3. **(Union bound in action.)** Let $X_1, \dots, X_n$ be random variables on a common probability space, and suppose $P(|X_i| > t) \leq e^{-t^2/2}$ for each $i$. Use Boole's inequality to bound $P(\max_i |X_i| > t)$ and discuss when the bound is tight (i.e. for what dependence structure across the $X_i$).
